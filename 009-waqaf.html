Waqaf

Pendahuluan

  1. Waqfun: berhenti bacaan, dengan niat nak sambung.
  2. Qat'un : berhenti bacaan, tanpa niat nak sambung.
  3. Sebab-sebab waqaf:
    1. Ikhtibari : ketika ujian, proses pembelajaran.
    2. Idhthirori : bila terpaksa (bersin, habis nafas, letih, lupa, menangis, batuk).
    3. Intizhori : untuk ulang dan baca guna riwayat lain.
    4. Ikhtiyari : memang sengaja nak berhenti.
  4. Ikhtiyari dipecahkan lagi kepada:
    1. Tam : sempurna.
    2. Kafi : memadai.
    3. Hasan : baik.
    4. Qabih : buruk.
  5. Beza:
    Waqaf Ada kaitan dengan kalimah selepasnya
    Lafaz (tatabahasa) Makna
    Tam ❌ ❌
    Kafi ❌ ✅
    Hasan ✅ ✅
    Qabih ✅ ✅

    Beza hasan dengan qabih, ialah qabih merubah makna ayat.
    1. Hasan: boleh berhenti, cuma kena ulang semula.
    2. Qabih: tak boleh berhenti.
